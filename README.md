# 30 Bash Script Examples

[The original resource of source code](https://linuxhint.com/30_bash_script_examples/)

## Executing ways

The first way:

```
$ bash <bash-script>
```

The second way:

```
$ chmod a+x <bash-script>
$ ./<bash-script>
```

For example:

```
$ bash 01_first.sh
Hello World
$ chmod a+x <bash-script>
$ ./01_first.sh
Hello World
```
